# Coffee Shop Website

A modern full-stack Coffee Shop web application built using **Next.js**, **Node.js**, and **MongoDB**.  
This project allows users to explore the coffee menu, view product details, and contact the shop — while the admin can manage products through a secure dashboard.

## Features
-  User Authentication (Google & Email login)
-  Admin Panel for managing menu items (Add, Update, Delete)
-  Dynamic Menu rendered from MongoDB
-  Contact and About sections
-  Fully Responsive UI using Tailwind CSS
-  Smooth navigation with scroll effects
-  Product images stored securely (Cloudinary integration)
-  API routes built with Next.js server components

## Tech Stack
- **Frontend:** Next.js 14, React, Tailwind CSS
- **Backend:** Node.js, Next.js API routes
- **Database:** MongoDB
- **Auth:** NextAuth.js (Google Provider)
- **Deployment:** Vercel / Render

## Admin Features
- Login via Google Authentication
- Add new coffee products with name, image, description, and price
- Delete existing items
- Toast notifications for actions
- Realtime UI updates on actions

## Purpose
This project was created to demonstrate a modern full-stack web app workflow — from design to database integration — suitable for learning, portfolio presentation, or small business deployment.
